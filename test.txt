1)cd "путь к директории"
2)~ означает домашнюю директорию, . текущую директорию, .. родительскую директорию
3)удаляет файл/папку/или директорию с дополнительными параметрами
4)команда перемещеия между директроями
5)команда копирования файла, или дириктории с дополнительными параметрами(дальше с "дп")
6)команда перемещения файла, или директории с дп, и команда для переименовывания файлов
7)mv "старое имя" "новое имя"
8)mv "имя файла" "путь"/
9)команда рекурсивно используется ко всем файлам в директории
10)команда ls будет работать в обратном порядке, остальные команды работают рекурсивно
11)распределённая система управления версиями
12)создание нового репозитория git
13)get branch "название ветки"
14)git checkout "название ветки"
15)Это слияние коммитов двух веток произвоящий новый коммит, выполняется командой git merge "название второй ветки", сливаются ветка из которой была использована команда и указанная ветка.
16)Удаление отдельных отсеживаемых файлов или директорий в git
17)Операция безопасной отмены изменений, вместо прямого удаления файлов создаёт коммит который и отменяет изменения.
